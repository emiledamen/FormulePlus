Voeg de volgende dependency toe aan je package.json en deploy opnieuw:

Dependencies-sectie:
  "@auth/prisma-adapter": "^1.0.7"

Voorbeeld (alleen illustratief):
{
  "dependencies": {
    "@auth/prisma-adapter": "^1.0.7"
  }
}

Je hoeft verder niets te wijzigen. Na toevoegen: `npm install` (lokaal) of laat Vercel het automatisch installeren bij de volgende build.
